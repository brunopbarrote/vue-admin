# vue2-ssr-example

Vue 2.0 Server-Side Rendering Example. [Read more](http://csbun.github.io/blog/2016/08/vue-2-0-server-side-rendering/).

## Installation

```bash
npm install
```

## Development

```bash
npm start
```

Hit [http://127.0.0.1:3000/](http://127.0.0.1:3000/) on your favorite browser.
